Rules Of The Game of Set

  • Hello, and welcome to the Game of Set!
  • To play the game you will select a set of 3 cards
  • Each card has 1 of 3 value
  • 1 of 3 colors
  • 1 of 3 shapes
  • and 1 of 3 shadings
  • For level 1 choose a set of 3 cards based on values
  • For level 2 choose a set of 3 cards based on values and shapes
  • For level 3 choose a set of 3 cards based on values, shapes, and colors
  • For level 4 choose a set of 3 cards where each attriibute is either the SAME or DIFFERENT for all 3 the cards.
  • Choose "Statistics" to get statistics for the computer player
  • Goodluck!
